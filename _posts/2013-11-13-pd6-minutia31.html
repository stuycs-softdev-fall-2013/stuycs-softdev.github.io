Creating a Repository

Mr. Z then showed us how to create a new repository on Github for each group.

  1. Go to new repository and name it proj2-pd6-groupNumber(01-08)-groupName.
  2. Make the repository public.
  3. Initialize repository with README.
  4. Add .gitignore: Python.
  5. Go to settings, then collaborators and add stuycs-softdev/everyone to teams (this means that everyone in both periods will have access to your repository).
