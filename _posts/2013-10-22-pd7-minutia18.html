CSS Stuff

  • Twitter Bootstrap is probably the most famous CSS framework. It was used to make twitter, then released to the public as an open source project.
  • Other famous CSS frameworks/libraries include ZURB FOUNDATION (which has a yeti for a mascot), purecss.io, 960 grid, minimalist grid, and others.
  • Twitter Bootstrap and Zurb are large, very inclusive libraries. The downside to this is that your web application will look a lot like everyone else's web applications.
  • CSS files often come in 2 versions: a regular version, which is readable to a normal human, and a min version, which is very compact in order to speed up the transfer of information.
  • Bootstrap is well documented; visiting the website http://getbootstrap.com/ can teach you how to use this awesome framework.
