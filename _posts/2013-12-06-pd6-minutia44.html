Good Morning

Good Morning! Website
Good Morning! Source Code

This project was created by Dorit Rein, Victoria Greene, Jason Luo and Raymond Lam, and aimed to make it easy to view the most important information in the morning. It takes into consideration MTA Information (screenscraped from a constantly-updated .txt file), the weather (from a Weather Underground API), and one's events on a calendar that Victoria created herself (and calls her baby). The group used a Pure CSS library in order to create their website.

The calendar was a particularly noteworthy part of this project, as Mr. Z even said that making a calendar from scratch is a pain. However, he pointed a section in the calpage.html in which there are a bunch of if-else statements and advised us against this kind of coding, and to use a table instead. Nevertheless, it was a great project that can go in many directions.

Artificial Intelligence Dating Site / Fake Profile Generator

Profile Generator Website
Profile Generator Source Code

This project was created by Jasper Lu, Arfan Rehab, Jason Zhen, Stanley Zheng, and generated fake profiles upon request. Originally it was going to be an AI Dating site, but due to time constraints, they only got around to generally a variety of fake profiles who became friends based on their assigned hobbies and interests. In order to do this, the group used an About.me API and screenscraped from a Twitter profile generator.

The styling was really cool here with a responsive front page and a cool loading symbol, which the group explained using their source code. A particularly hilarious part of the project's demostration was the fact that many of the fake profiles had generated the same profile picture.
