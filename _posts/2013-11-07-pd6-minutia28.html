Getting data from the internet

One method of getting data from a website is by screen scraping. This involves going to the actual website and retrieving all of the HTML from the site.
Python Example:
import urllib2

url = "http://www.nytimes.com"
request = urllib2.openurl(url)
result = request.read()
print(result)
Drawbacks
  • JavaScript changes to elements will not appear in the HTML
  • To retrieve data you need to go through the whole mess of HTML tags just to find what you need

BeautifulSoup

BeautifulSoup provides the tools to more easily retrieve data from a site.
Python Example:
from bs4 import BeautifulSoup
import urllib2
url = "http://www.nytimes.com"
soup = BeautifulSoup(url)

print(soup.prettify()) # Formats the HTML in a nicer manner
print(soup.title) # Returns only the title element from the page
for a in soup.find_all('a'): # loops through all the <a> tags in the HTML
  print(a.get_text()) # returns only the text of the elements

There has to be something easier though, right?

API

Most websites that contain data provide APIs that only output data rather than a whole entire webpage which helps web developers retrieve that data much easier.
RESTful APIs allow developers to enter arguments into a GET request and have an output, normally JSON. JSON can easily be converted directly into a dictionary which allows for much easier use than screen scraping.
