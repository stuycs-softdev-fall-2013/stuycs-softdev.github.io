Some prehistoric facts about HTML during the Medieval Times (1990s)

  1. When HTML was first used, the terminal was displayed in black and white, not gray scale.
  2. Typically, the terminal did not support some features such as bold. This meant that the output of bold had to be viewed in the browser.

It was also mentioned that when websites, these days, have a similar layout to magazines. When using historic HTML to code headers, for example, one would have to change each header individually. Due to this lack of efficiency, CSS(Cascade Style Sheets) Styling has been introduced.

CSS Styling

  • Mr.Zamansky introduced one way of using style sheets by defining them within the style tags.
  • CSS Styling is separate from the HTML
  • You can manipulate style using classes by doing .nameofclass{feature}, for example: you would use .odd{background-color:gray;} to set whatever that has class odd to a gray background NOTE: the color can be defined using hexadecimal (color:#FF0000)
  • You can also use divide(div) tags which separates code to their own specific area
  • It is RECOMMENDED to search up the CSS menu
  • Bootstrap is a website that gives css code to easily create user interfaces. Bootstrap essential gives you full functionality of different types of grids, tables, buttons, and images without you doing the hard work!
